Create Comment Box :

box e kichu likhe button e click korle input field er text ta upore comment hisebe add hobe & textbox ta abr blank hoye jabe

Steps -- >

  • 1. get the element & add 'EventListener'.
    ex : document.getElementById('button-post').addEventListener('click', function(){})
  • 2. store the value of textarea in a variable .
    ex : const commentBox = document.getElementById('new-comment');
  • 3. Set the comment text in a variable
    ex : const newComment = commentBox.value;
  • 4. Set the comment inside the comment container
    ex : const commentContainer = document.getElementById('comment-container');
  • 5. Create a paragraph tag
    ex : const p = document.createElement('p');
  • 6. Set the comment inside the p tag
    ex : p.innerText = newComment;
  • 7. push it
    ex : commentContainer.appendChild(p);
  • 8. Clear the comment box.
    ex : commentBox.value = '';
